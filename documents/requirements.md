# Веб-сервис: онлайн-регистрация в салон.  

## Задача проекта:
Необходимо создать веб-сервис для онлайн-регистрации в салоне, который позволит любому желающему пройти небольшую регистрацию на сайте, чтобы в удобное для себя время записаться на услугу.

***
## Неспецифические требования
### Основные функции приложения:
+ Запись к одному/нескольким специалистам на свободные места (дата и время)
+ Краткий обзор услуг, предоставляемых салоном, а также их цен
+ Просмотр новостей, акций и выходных дней
+ Подтверждение о записи в салон на электронную почту 
+ Небольшой календарь, для возможности сориентироваться по дате


### Режимы функционирования:

+ Окно регистрации записи к специалисту
+ Окно, содержащее календарь
+ Просмотр ленты информации


### Работа с веб-сервисом

Перейдя на веб-сервис онлайн-регистрации в салон, пользователь может просмотреть необходимую информацию, профиль мастера, его заслуги и отзывы о нем, а также оставить отзыв самому и записаться на услугу в поле регистрации, для этого не нужно регистрироваться на сайте, просто необходимо записать информацию в поля для заполнения. По окончании заполнения, пользователь может выбрать дополнительную услугу. После записи на услугу(и) пользователю приходит подтверждение на электронную почту. 
#### Окно онлайн-регистрации содержит поля:
+ ФИО*
+ Номер телефона*
+ Электронная почта*
+ Выбор услуги*
+ Выбор специалиста - дает возможность для записи к определенному специалисту
+ Выбор даты* - позволяет выбрать свободную дату в течении месяца.
+ Выбор времени* - позволяет выбрать свободное время на нужную дату. Выбор уже имеющегося времени, так как время ставится по согласованию с мастером. 
+ Цена - автоматическое поле, которое генерирует примерную цену, в зависимости от предоставляемой услуги.
+ Кнопка "Добавить услугу" - дает возможность выбрать несколько услуг, не проходя регистрацию заново.
+ Кнопка "Регистрация" - выводит сообщение о том, что она была успешна пройдена и отправляет подтверждение на электронную почту, если все поля корректно заполнены.

Поля под "*" обязательны для заполнения.
#### Профиль сотрудников
Со стороны сострудников салона в веб-сервесе должен быть предусмотрен личный кабинет. Мастера и администратор(ы) должны пройти регистрацию. Войдя в свой личный кабинет, они могут заполнить информацию о себе. Также они будут получать уведомления о записи, а мастера еще и отзывы о их работе. Администраторы имеют возможность распределять время работы мастеров. 

#### Профиль мастера выглядит следующим образом: 
+ Поле ФИО*
+ Поле Фотография 
+ Поле Пол и возраст*
+ Поле Вид услуги*
+ Поле Образование*
+ Поле Стаж*
+ Поле Дипломы/конкурсы и тд.
+ Экран "Отзывы" - мастерам салона доступен только к просмотру и комментариям, оставлять отзыв могут только пользователи. 
+ Экран "Уведомления" - позволяет просматривать записи к мастерам самим специалистам, если их выбрали при онлайн-регистрации.


#### Профиль администратора
+ Поле Должность*
+ Поле ФИО*
+ Поле образование*
+ Экран "Отзывы" - администраторам салона доступен только к просмотру и комментариям, оставлять отзыв могут только пользователи.
+ Экран "Уведомления" - позволяет просматривать все записи к мастерам.
+ Экран "Добавление/Редактирование даты и времени" - доступен только администраторам салона, они управляют временем для записи на услугу, по согласованию с мастером.

## Специфические требования

+ ***Корректность*** - данный веб-сервис должен работать корректно, отображать занятые даты и время, чтобы не было возможночти записаться на одну и ту же услугу в одно время.
+ ***Практичность*** - веб-сервис должен быть понятен любому пользователю, чтобы выполнять запросы и находить необходимую информацию было легко.
+ ***Эффективность*** - веб-сервис должен работать без задержек/ошибок. Оперативно выдавать свободное и занятое время для записи. Отклик системы не должен превышать 10 секунд с момента передачи запроса.
+ ***Целостность*** - обезопасить приложение от нарушения целостности личных данных пользователей и некорректного доступа.
+ ***Живучесть*** - для работы  веб-сервиса необходимо иметь подключенное Интернет-соединение. При возникновении проблемы корректно завершить обработку запроса и вывести сообщение о возникших неполадках.
+ ***Возможность повторного использования*** - к веб-сервису можно обращаться неоднократно.
